---+ HelpXMLPlugin

This plugin allow you to create a help.xml file from a TWiki page. It create the help.xml page using the TOC tags in TWiki

---++ Syntax Rules

   * =%<nop>HELPXML{"HelpOnLine" url="helpcontent/helponline.html"}%=

---++ Plugin Settings

   * Set DEBUG = 0
